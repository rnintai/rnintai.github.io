# 웹
## 웹이 무엇인가?
인터넷이라는 전세계적 망을 활용하여, HTML이라는 용어로 웹 페이지 데이터를 주고 받는 환경이다.  
자세히 말하면 웹 브라우저와 웹서버가  
HTTP라는 규약을 이용하여 HTML언어로 된 웹페이지를 주고받는 것.  

# 서버와 클라이언트
1. 웹브라우저(Client)가 서버에 서비스를 요청함.
2. 서버는 이에 응답한다.
