# 자바스크립트 태그
`<script>` 와 `</script>`의 태그를 통해 표현한다.
```
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8"/>
    </head>
    <body>
        <script>
            alert('Hello world');
        </script>
    </body>
</html>
```
